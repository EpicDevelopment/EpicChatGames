
# Epic ChatGames

Basic chatgames plugin for Minecraft version 1.20.4


## Usage/Examples

```
/chatgames start <math/fastest/scramble>
/chatgames add <word>
/chatgames list
```


## Dependencies


 - [Spigot](https://getbukkit.org/get/272245e4f948b0a66b0b4c34dfa27c49)
